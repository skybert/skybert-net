My First Mac

I got my first Mac today. A shiny piece of metal called Macbook Pro 7.1. I've gotten quite a lot of reactions to this, so I thought I'd put up a FAQ :-)

Frequently Asked Questions

What?!! I thought you were a die hard Linux user? Have you finally realised that Linux is a dead horse?
Relax, I'll still be running Linux. In fact, I've already set up a dual boot with Debian GNU/Linux.
So why did you get a Mac, then if you're planning to run Linux on it?
I like the hardware. It has good build quality, good camera, good battery life and good specifications overall.
Why do you bother installing Linux when Mac OS is Linux too?
First of all, Mac OS is not Linux. It's based on FreeBSD Unix, something which is very different. Secondly, Linux is so much more powerful for a system administrator and programmer as it gives you immediate access to all popular server components and operating system internals. Thirdly, running Linux allows me to use different desktops and window mangers that make me far more effective than using the Mac OS clicky clicky interface.
Do you plan to use Mac OS X at all?
Yes, whenever I'm to connect fancy external multimedia peripherals, I'll switch to Mac. Video and audio are fields where Mac is wonderful, it "just works" (at least in my experience). Also, it's great to learn a new operating system, one that many people are very excited about!

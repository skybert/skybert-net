#+title: A simple example of using JPA

Here, we've got a really simple domain object with one ID field and a
name.

#+begin_src java
import javax.persistence.Entity;
import javax.persistence.Id;

@Entity
public class Fruit {
  @Id
  private Integer id;
  private String name;

    public String getId() {
        return id;
    }

    public void setId(String id) {
        this.id = id;
    }
}
#+end_src

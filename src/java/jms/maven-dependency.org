#+title: Using JMS from Maven

I was stunned to find that =javax.jms:jms= wasn't available in the
[[http://search.maven.org/#browse|2127833746][Public Maven Repo]], only the POM was.

I got around this by adding the public JBoss maven repo to my
=settings.xml=:
#+begin_src text
<repository>
  <id>jboss-public</id>
  <name>JBoss.org Maven repository</name>
  <url>https://repository.jboss.org/nexus/content/groups/public</url>
</repository>
#+end_src

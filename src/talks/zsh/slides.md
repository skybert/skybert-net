# Forgetting a pipe

```text
$ bash
$ echo hi |
> wc
```

```
$ zsh
$ echo i |
pipe> wc
```

---

# Forgetting a quote
```text
$ bash
$ echo 'hi 
> '
```

```
$ zsh
$ echo 'hi
quote> '
```

# Hot deploying JavaScript from Emacs to Firefox
First off, install the excellent
[mozrepl](https://addons.mozilla.org/en-us/firefox/addon/mozrepl/)
plugin for Firefox.

Then, install its counterpart for emacs:

    M-x package-install moz

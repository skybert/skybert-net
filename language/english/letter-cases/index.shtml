Letter Cases

Upper Case

ID is always upper cased (not 'id' and 'Id'). You may have a function in your code that's called getId(), but when you write human readable texts, it should be upper cased.

This is also true for all abbrevations, they should always be upper cased: XML, REST and HTTP

Capitalisation

Product names are always capitalised (remember, they're just like people's names), hence; Community Engine, Struts and Ant.

All table, chapter and section headings are capitalised.

The first letter after a comma (',') is not to be capitalised like is the case after a period ('.').

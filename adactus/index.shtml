Seeing which video profile/variant Adactus/Mobilize/VMEO/Diactus finds for the web clients, see the output of: adactus.diactus.di.adpatation.engine.AdaptationProfileEngine: 

$ tail -f /var/diactus/log/diactusDebug.log | grep adactus.diactus.di.adpatation.engine.AdaptationProfileEngine

Unrecognised devices

To see which devices your VMEO isn't able to find a profile for, have a look in UnkownDeviceInfo.log: 

$ tail -f /var/diactus/log/UnknownDeviceInfo.log

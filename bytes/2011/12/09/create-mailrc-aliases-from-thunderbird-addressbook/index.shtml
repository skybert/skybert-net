create mailrc aliases from thunderbird addressbook

I've recently switched back to using Gnus instead of Thunderbird for my email at work and I needed my addressbook. To do this, I chose Tools -> Export in Thunderbird's addressbook and chose CSV as the format.

I then created this wee script to create mail aliases in my .mailrc for my addressbook: 

cat addresses-from-thunderbird.csv | while read f; do
     if [ $(echo $f | grep "@" | wc -l) -lt 1 ]; then
         continue
     fi
     
     name=$(echo $f | cut -d',' -f1 | sed 's/ /\./g')
     email=$(echo $f | cut -d',' -f5)
     echo alias $name $email >> $HOME/.mailrc
 done

The entries generated are on the form: 

alias name.lastname email@address.com

With the entries in .mailrc, Gnus, VM and any other traditional Unix mail client can expand the email addresses for me by using the alias short hand.

First impressions with puppet

These are some of my first impressions using Puppet 2.6.2 on Debian.

Debug messages are written to standard out

Defaultwise debug messages, if specified in /etc/default/puppet are written to standard out. Argh! How about following all/most other Unix components and log this to a file in /var/log, Puppet?

What's worse, thoug, is that if I specify a log file, which according to the man page should instead put it in a log file, it still writes the messages to standard out. The difference being that it now also writes it to a log file.

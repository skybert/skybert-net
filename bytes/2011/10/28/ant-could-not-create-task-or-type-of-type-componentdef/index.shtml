Mysertious Ant error could not create task or type of type componentdef

I was getting this error and couldn't make heads or tails of it. The Ant libraries seemed to contain the task at hand and it just didn't make any sense: 

[INFO] An Ant BuildException has occured: The following error occurred while executing this line:
jar:file:/home/torstein/.m2/repository/org/apache/maven/plugins/maven-antrun-plugin/1.6/maven-antrun-plugin-1.6.jar!/org/apache/maven/ant/tasks/antlib.xml:24:
The following error occurred while executing this line:
jar:file:/home/torstein/.m2/repository/org/apache/ant/ant/1.8.1/ant-1.8.1.jar!/org/apache/tools/ant/antlib.xml:37:
Could not create task or type of type: componentdef.

Ant could not find the task or a class this task relies upon.

It turned out I had to use Maven 3 instead of Maven 2 and also set the MAVEN_HOME environment variable: 

# .basrhc
export MAVEN_HOME=/opt/apache-maven-3.0
export PATH=$MAVEN_HOME/bin:$PATH

With this in place, the project built just fine.

Man, it's hard to track down this things when the error message is so far from the truth.

fixing the phionvpn debian packages

After installing the phionvpn Debian package, I always get errors like these when using apt-get: 

  dpkg-query: warning: parsing file '/var/lib/dpkg/status' near line 15522 package 'phionvpn':
 error in Version string 'R8-SP2': version number does not start with digit

The proper fix is of course to fix the control file inside the Debian package and rebuild it, but the quick fix to tidy this mess up after installing this is to:

Edit /var/lib/dpkg/available and /var/lib/dpkg/status, search up the entries for Phion VPN and change from: 

R8-SP2

to 

8.0-SP2

That's it. dpkg and apt-get doesn't complain anymore.

iBus Not Working in GTK Applications

I hade the problem that ibus wasn't working in GTK applications on my GNOME3 desktop. Funnily enough, it worked in Qt applications like the Opera web browser.

After a lot of digging it turned out that it was because I both had the ibus-gtk and the ibus-gtk3 package installed. Removing the ibus-gtk resolved the issue: 

# apt-get remove ibus-gtk

I this will save some fine folks for a few minutes or hours of error hunting!

Cannot change locale

I was getting these errors whenever I was runnign anything which tried to read my locale: 

-bash: warning: setlocale: LC_ALL: cannot change locale (en_GB.utf8)

The remedy (on Ubuntu 11.10), was to generate the locale: 

# locale-gen en_GB.utf8

On a related note; on Debian, you can select all the locales youw ant from a nice curses based dialogue by running: 

# dpkg-reconfigure locales

Sadly, this didn't work on Ubuntu, hence I had to hunt down locale-gen.

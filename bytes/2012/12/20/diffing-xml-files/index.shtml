Diffing XML files

Let's say you've got these two XML files: 

$ cat test*xml
<xml><arne tore="hallo" lise="hei"/></xml>
<xml><arne lise="hei" tore="hallo"/></xml>
$

Now, if you diff them as regular files, you will see that they are different, even after you've pretty printed them with the ubiquitous xmllint: 

$ diff <(xmllint --format test.xml) <(xmllint --format test2.xml)
3c3
<   <arne lise="hei" tore="hallo"/>
---
>   <arne tore="hallo" lise="hei"/>
$

Now, this may very well be exactly what you want. However, if you're thinking of the differences XML wise, there are none between the two XML files, it's the same elements and the same attributes. Here, xml_pp helps us the last bit of the way: 

$ diff <(xml_pp test.xml) <(xml_pp test2.xml)
$

There! No difference and all is good.

Blistering Fast Installation - I

On Debian based systems, install the escenic-content-engine-installer package: 

# curl -s http://apt.vizrt.com/repo.key | apt-key add -
# echo "deb http://apt.vizrt.com unstable main" > /etc/apt/sources.list.d/vizrt.list
# apt-get update
# apt-get install escenic-content-engine-installer

Blistering Fast Installation - II

Write your ece-install.conf file based on the Escenic Content Engine Installer Guide or base it on any of the pre-made examples configuration files.

Blistering Fast Installation - III

With ~/ece-install.conf in place, let the installer do all the work for you: 

# ece-install
